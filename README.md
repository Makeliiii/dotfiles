# Dots
"Template" is from the [EndeavourOS SwayWM setup](https://github.com/EndeavourOS-Community-Editions/sway) with, pretty much, a complete overhaul from me.

## Deetz
* OS: [EndeavourOS](https://endeavouros.com/)
* WM: [SwayFX](https://github.com/WillPower3309/swayfx)
* Bar: [Waybar](https://github.com/Alexays/Waybar)
* Terminal: [kitty](https://github.com/kovidgoyal/kitty)
* Browser: [Firefox](https://www.firefox.com)
* Editor: [NvChad](https://nvchad.com/)
